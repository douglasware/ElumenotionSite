Using http-server

After executing BuildDocs.ps1 the BuildOutput folder will contain a complete static website. In the case of this sample, BuildOutput is preloaded. There are numerous options for serving content located in a folder over Http.
One recommended choice is located in NPM at http-server
Note that this option requires npm and it’s dependencies

Hot Reloading

You don’t have to stop the web server to run the build script. However it is highly recommended that you open developer tools and disable caching!
