Access Control

All EY users should have access, but they must manually sign-in at this time. The easiest way to do this is to create and use a favorite to GitHub SSO: https://github.com/enterprises/ey/sso. If you share a link to this site to an EY user, and they see the 404 error page, the most likely reason is that they are not signed into GitHub Enterprise.
